# 청년 지방 이주 시뮬레이션 웹 서비스 - 카카오임팩트 협업

## 프로젝트 소개
지방 이주를 고민하는 청년이 떠나기 전, 그 지역을 먼저 살아볼 수 있는 웹 서비스입니다.
5가지 질문으로 이주 유형과 지역을 추천받고, 추천된 지역에서 3일간 아침·점심·저녁 미션을 수행합니다.
카카오 로드뷰로 실제 골목을 걷고, 체험이 끝나면 Claude API가 자동으로 맞춤 이주 리포트를 생성합니다.

## 역할
- 서비스 기획 및 팀 리드
- 사용자 답변을 점수화해 이주 유형과 지역을 추천하는 알고리즘 설계
- Claude API 기반 이주 리포트 자동 생성 기능 구현
- Kakao Maps API 로드뷰 미션 연동

## 기술 스택
React · TypeScript · Tailwind CSS · Kakao Maps API · Claude API · Vite
