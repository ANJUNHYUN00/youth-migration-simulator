# 마당 꾸미기 시스템 — 팀 공유용 논의 문서

작성일: 2026-06-12
작성자: 현지 + Claude

---

## 1. 우리가 풀고 있는 질문

> "내 집/마당을 어떻게 자라게 할까?"

귀촌 시뮬레이션의 핵심 정서 중 하나 — **시간이 지날수록 내 공간이 풍성해진다**는 감각.
이걸 어떤 메커니즘으로 구현할지 두 갈래에서 고민 중.

---

## 2. 큰 갈래 두 개

### 갈래 A. 집을 완성적으로 지어 나간다 (원래 안)
- 처음엔 빈 터 또는 골조만.
- 미션을 깰수록 집 자체가 단계적으로 완성됨 (지붕 → 벽 → 창 …).
- 게임 빌더 톤.

**장점**
- "성장"이 매우 직관적. 진행률 100% = 집 완성, 라는 클리어한 시각.

**단점**
- 자유도 0 — 사용자가 만지는 게 아니라 자동으로 자람.
- 집이 다 지어지면 그 뒤가 비어버림.
- 귀촌의 정서("내가 가꾼다")보다 "체크리스트" 톤에 가까워질 위험.

---

### 갈래 B. 처음부터 집을 주되, 미션 보상 아이템으로 마당을 꾸민다 (현재 검토)
- 첫 진입부터 집·마당은 있음.
- 미션 완료 → 꾸미기 아이템 획득 → 다음 날 마당에 반영.
- 한 곳에 오래 머물수록 그 사람만의 마당이 됨.

**장점**
- 사용자가 주도. "내가 만들었다"는 감각.
- 미션과 보상의 연결이 또렷함 (이 화분 = 어느 미션의 기억).
- 끝이 없음 — 8지역 다니면서 계속 풍성해짐.

**단점**
- 구현 부담이 갈래 A보다 큼.
- "지금 빈 마당 = 미션 안 한 사람"이라 첫인상이 약할 수 있음 → 초기 시드 아이템 필요할 수도.

---

## 3. 우리의 선택 — 갈래 B

방향은 B로 가기로. **이제 B 안에서 또 두 가지가 고민됨.**

---

## 4. B 안에서의 두 옵션

### 옵션 B-1. 꾸미기 아이템을 **별도 트랙**으로
- 기존 **기념품**(JourneyScreen 일차 탭의 3x3 그리드) 그대로 유지.
- 미션 보상이 **두 종류**가 됨: 기념품 1 + 꾸미기 아이템 1 (혹은 미션마다 어느 한 쪽).
- 두 트랙이 각자 따로 모임.

**장점**
- 두 종류가 모이는 만족감. "오늘 두 개 받았다" 효과.
- 기념품(기억) vs 꾸미기(현재 공간) 의미가 깔끔히 분리.

**단점**
- 사용자에게 보상이 두 군데로 흩어짐 → 알림·UI 복잡도 증가.
- 비슷한 게 두 개라 산만하게 느낄 수도.

---

### 옵션 B-2. **둘을 하나로** — 기념품 = 꾸미기 자재
- 미션 보상이 **하나만** 떨어짐. 그게 동시에 기념품이자 꾸미기 자재.
- JourneyScreen 기념품 그리드는 → 마당 꾸미기 화면이 대체 (혹은 같은 데이터를 두 군데서 봄).

**장점**
- 단순함. "미션 깼다 → 보상 1개 → 마당에 둔다" 한 흐름.
- 한 아이템에 의미 두 겹 (기념 + 기능).

**단점**
- 기념품 페이지(현재 디자인)와 마당 페이지 중 하나는 사라지거나 형태가 크게 바뀜.
- 두 트랙 모으는 만족감은 없음.

---

## 5. 꾸미기 아이템 카테고리 — 5종 제안

미션 보상으로 떨어지는 아이템은 **"이게 어디서 왔는지 기억나는 물건"** 이라야 의미가 산다.
한옥+파스텔 마당 톤과 미션 유형에 매핑되는 5개 카테고리:

| 카테고리 | 미션 매칭 예 | 마당에서의 의미 |
|---|---|---|
| 🌱 **화분** | 텃밭/정원/식물 체험 | 식물이 늘어남 — 가장 자주 떨어지는 기본 보상 |
| 🪨 **돌·디딤돌** | 산책/숲길/마을 둘러보기 | 길이 깔리고 발 디딜 곳 생김 |
| 🪑 **평상/의자** | 차 마시기/마을 사람 만나기 | "쉴 자리"가 생김 — 정서적 의미 |
| 🏮 **등불·풍경** | 야간 활동/별 보기/소리 미션 | 빛·소리 — 마당이 살아있다는 느낌 |
| 🐾 **작은 친구** | 동물 만나기/이웃 미션 | 길냥이·새 등 마스코트 — 희귀 보상 |

### 왜 5종인가
- 미션 1개당 보상 1개로 치면, 8지역 × 며칠치 미션 ≈ 카테고리당 평균 3~5개 모임.
- 빈약하지도 부담스럽지도 않음.

### 줄이거나 늘릴 여지
- **줄인다면**: 화분/평상/등불 3종이 정서 핵심.
- **늘린다면**: 빨랫줄, 우물, 깃발, 장독, 새집 같은 한옥 마당 소품.

---

## 6. 꼭 메모해둘 만한 디테일

### 각 아이템에 메타데이터 붙이기
한 아이템이 곧 한 미션의 기억이 되려면 최소 이 정보가 필요:

```
{
  이름,         // 예: "강화 차밭의 작은 평상"
  카테고리,     // 화분/돌/평상/등불/친구
  어디서 왔는지: {
    지역,       // 강화도
    미션,       // 차밭에서 차 한 잔
  },
  획득일,       // Day 3
}
```

→ 마당에 놓을 때 카피로 자동 노출:
"강화도 차밭 미션에서 받은 평상"
이런 한 줄이 감정 트리거가 됨.

### 다음 날 반영의 의미
"미션을 마치면 작은 흔적이 하나씩 쌓여요" → 즉시 반영하지 않고 **다음 날** 마당에 등장.
- 게임적 보상(즉각 도파민)이 아니라 **귀촌의 시간 감각**과 맞물림.
- "잠들고 일어나면 마당이 어제와 달라져 있다" — 이 정서가 핵심.

### 빈 마당 첫인상 보완
B의 약점인 "빈 마당"을 보완하려면:
- 시드 아이템 1~2개 (예: 입주 환영 화분) 깔아두기.
- 또는 첫날 미션 1개를 튜토리얼처럼 짧게 → 첫 보상 빠르게 가져가게.
