**작성일**: 2026-06-01
**대상 미션**: 5개 (강화-시장 / 강화-갯벌 / 영월-별마로 / 태안-만리포 / 진도-운림산방)

## 시트 상단 메모 / 주의사항

1. **별마로 미션은 도보권 레지던스가 사실상 존재하지 않습니다.**
   별마로천문대는 봉래산(해발 799.8m) **정상**에 있고, 등산로 입구에서 정상까지 도보 ~50분(나무계단 20분 + 능선 30분). 천문대 자체 숙박시설(천문과학교육관)은 **학교 단체 천문 프로그램 전용**으로 일반 관람객 단순 숙박 불가(영월시설관리공단 안내). 영월읍내 숙소(예: 탑스텐 동강시스타)도 직선거리 3.6km로 도보권 ❌. → 후보 1개(미확인)만 등재. 사용자 추가 검토 필요.

2. **강화 시장/갯벌 두 미션은 섬 내 위치가 정반대**입니다.
   - 풍물시장: 강화읍 북쪽 (중앙로 17-9)
   - 동막해변: 화도면 남쪽 (해안남로 1481, 약 22km 떨어짐)
   → 두 미션을 **하나의 레지던스로 묶지 않고** 각각 따로 후보를 잡았습니다.

3. **도보 거리는 후보지 공식 페이지 / 운영사 표기 기준**이며, 카카오맵·네이버맵 직접 길찾기 검증은 사용자가 시연 직전 한 번 더 해주시기를 권합니다 (특히 "도보 N분" 표기가 차량 기준일 가능성 있는 경우 notes에 명시).

4. **walking_distance_to_destination 단위**: 직선 또는 도로 m, 또는 "도보 X분" 표기. 정확한 도보 거리가 검증 안 된 셀은 `미확인`.

5. **운림펜션 (진도)**: visitkorea 안내문에서 "운림산방·첨찰산 정류장에서 운림산방까지 도보 약 440m" 라는 표현이 등장. 운림펜션은 의신면 운림산방로 도로 상에 위치 — 같은 도로 인접 (사천리 운림산방 권역). 정확한 펜션-운림산방 도보 거리는 사용자가 카카오맵으로 한 번 더 확인 권장.

## 미확인 셀이 많아 사용자 추가 정보 요청이 필요한 미션

1. **yeongwol-stars (별마로 천문대)**: 도보권 일반 레지던스 부재.
   → 사용자 결정 필요: (a) 미션 컨셉을 "영월읍 동강 일대 → 야간 천문대 셔틀" 로 재설계, 또는 (b) 별마로 미션 자체를 다른 영월 미션(청령포·한반도지형·동강 래프팅 등)으로 교체.

2. **모든 미션의 `coordinates` 컬럼**: 카카오맵/네이버맵에서 위경도 확인 안 됨 (시설 단위 좌표는 보통 직접 지도 핀 확인 필요). 시연 전 디자이너/개발자가 지도 핀 좌표 추출 권장.

3. **walking_distance_to_destination** 의 "추정" 표시 셀: 카카오맵/네이버맵 도보 길찾기로 한 번 더 검증 권장:
   - 남문한옥 대명헌 ↔ 강화풍물시장
   - 동막 스파 펜션 ↔ 동막해변
   - 운림펜션 ↔ 운림산방
