# 공통 미션 정리 (9개)

모든 지역에서 공통으로 풀 수 있는 미션. 사용자가 선택한 지역에서는 **공통 9개 + 지역 4개 = 총 13개 미션**을 체험.

진행 방식 표기:
- 🗺️ `map-info` — 지도 이동 후 정보형 안내 (1~3 turn)
- 🗣️ `map-dialogue` — 지도 이동 후 NPC와 대화 (3~6 turn)
- 💬 `dialogue` — 바로 대화 (3~5 turn)
- 🔢 `numeric` — 대화 + 수치 입력 단계
- 📮 `mailbox` — 우편함 모달

분류 표기:
- 💡 생활현실형
- 🤝 관계형성형
- 🌿 감정/분위기형

## 미션별 상세 — 대화 깊이와 핵심 정보

### 1. 병원 접근성 확인 (`hospital`) · 6 turn · +5
**시나리오**: 동네 어르신이 길에서 만나 병원 안내
- turn 0: "어디 찾고 있어요?" → 병원/산책 분기
- turn 1: "병원이요? 12분 거리예요" → 걷기/버스 분기
- turn 2: 걷기 선택 → 약국 옆 + 30분이면 끝
- turn 3: 버스 선택 → 35분 배차, 차라리 걷기
- turn 4: 산책 선택 → 동네 한 시간이면 그려져요
- turn 5: 마무리 — 시간이 천천히 가는 동네

**제공 정보**: 도보 12분 / 850m / 응급실 24시간 / 약국 인접

### 2. 전통시장 물가 체험 (`market`) · 4 turn · +5
**시나리오**: 시장 반찬가게 사장님과 대화
- turn 0: 가격 안내 (시금치 1,500원, 오이 6,000원)
- turn 1: "30% 정도 싸요. 채소·해산물만"
- turn 2: 영업시간 — 오후 5시면 마감
- turn 3: 한 달 식비 15만원 차이 — 외식 줄이는 효과

**제공 정보**: 시장 vs 도시 가격 / 영업시간 / 한 달 식비 영향

### 3. 생활비 시뮬레이션 (`cost`) · 4 turn · +5
**시나리오**: 1년 차 이주자가 자기 데이터 공유
- 월세 25~40만, 관리비 5만, 식비 25~35만, 통신 8만 = 약 80만원
- 차 유지비 +15만원 (옵션)
- 절약형 65만 / 보통 80만 / 여유 110만

**제공 정보**: 항목별 비용 / 도시 대비 절반 / 차 유지비 옵션

### 4. 교통 접근성 확인 (`transit`) · 4 turn · +5
**시나리오**: 마을 안내원이 교통 정리
- 시내 버스 배차 35분, 막차 21:30, KTX 40분, 공항 1시간 30분
- 차 없으면 자전거+버스 조합
- 동네 안에서는 자전거로 다 닿음

**제공 정보**: 버스/KTX/공항 시간 / 차 vs 자전거 / 출퇴근 없는 장점

### 5. 하루 루틴 체험 (`routine`) · 4 turn · +8
**시나리오**: 레지던스 호스트가 하루를 같이 그려봄
- 아침 7시: 산책 / 차 한 잔 / 작업 시작 분기
- 점심: 직접 해 먹기 / 식당 / 샌드위치 분기
- 오후 5시: 자연스러운 약속 시간대
- 저녁: 주민 어울림 / 집에서 / 산책 분기
- 마무리: 밤 10시면 동네 잠듦

**제공 정보**: 동네 시간 흐름 / 사회적 리듬 / 수면 깊이

### 6. 하루 식비 기록 (`food`) · 3 turn · +5 (🔢 수치 입력)
**시나리오**: 동네 식당 사장이 입력값 비교
- 입력: 오늘 식비 (원)
- 평균: 도시 14,000원 / 이 동네 9,500원
- 비교 결과 자동 분기 (절약형/평균/풍족형)
- 마무리: 외식 줄이는 게 핵심

**제공 정보**: 식비 평균값 / 입력값 비교 / 한 달 환산 효과

### 7. 동네 가게 들르기 (`shop`) · 4 turn · +15
**시나리오**: 30년 된 분식집 이모와 첫 만남
- "새 얼굴이네요" → 추천/구경 분기
- 단골 30년 가게 — 동네 어르신 다 단골
- 천천히 봐도 OK — 인사만 해도 동네 사람
- 5번 오면 얼굴 / 다섯 번이면 이름 묻는 사이

**제공 정보**: 단골 문화 / 도시와의 친밀도 차이

### 8. 이주민 만나기 (`neighbor`) · 5 turn · +15
**시나리오**: 1년 차 이주자가 솔직한 이야기
- 첫 6개월 어땠나 / 지금 만족도 / 후회 분기
- 적응기 3개월 답답 → 4개월 차 동네 친해짐
- 농사 모임 추천 (참여/조용 분기)
- 첫 겨울 우울 — 마음 준비 필요
- 마무리: "천천히 한 사람씩"

**제공 정보**: 적응 기간 현실 / 관계 형성 속도 / 시즌별 감정

### 9. 우편함 — 주민 이야기 (`mailbox`) · 1 turn (모달) · +3
**시나리오**: MailboxModal로 분기 — 도착 화면의 우편함 모달과 동일
- 지역별 1장 편지 (`data/stories.ts` 참조)
- 매일 1장 (디자인 의도) — 데모는 한 번만

**제공 정보**: 지역 이주민의 짧은 일상 텍스트

## 적합도 가산 룰 (요약)

각 옵션마다 `traits: LifeStyleType[]`이 부착되어 있고:
- `traits[0]` === 거주지 matchType → **+2**
- `traits`에 포함만 → **+1**
- 무관 → **0** (페널티 없음)

공통 미션은 모든 라이프스타일 유형에 대해 어느 정도 가점 받을 수 있게 traits를 분산 배치했어요. 자세한 산정은 `agents/frontend/mission-scoring.md` 참조.

## 데이터 위치

- 공통 미션 정의: `frontend/src/data/missions.ts` → `commonMissions`
- 최종 미션: 같은 파일 → `finalMission`
- 우편함 편지: `frontend/src/data/stories.ts` → `storiesByResidenceId`
- 지역 미션은 별도: `frontend/src/data/regionMissions.ts` (📄 `region-missions.md` 참고)

미션 추가/수정 시 `Mission` 타입을 만족해야 함 — `id`, `title`, `icon`, `category`, `mode`, `reward`, `background`, `npc`, `dialogues`, (선택) `description`, `realRoadview`.
