## 1. 온보딩 및 기본 정보 설정 (Onboarding & Setup)

### 1.1. 스플래시 화면 (Splash)
- **화면 설명:** 서비스 진입 시 노출되는 브랜드 아이덴티티 화면
- **카피라이팅:** - 메인 타이틀: `바람을 짓다.`
  - 서브 타이틀: `조용히 영글어가는 당신의 세계`
- **시각 요소:** 우주나 궤도를 연상시키는 동심원 그래픽과 은은한 파스텔톤 오브제 배치

### 1.2. 시뮬레이션 지역 선택
- **화면 설명:** 사용자가 체험하고자 하는 가상의 귀촌 지역을 선택하는 화면
- **카피라이팅:** - 메인: `'쉼'이 필요했던 나에게! 내가 살고싶은 곳은 어디일까?`
  - 서브: `입력한 정보를 통해 더 실감나는 귀촌 라이프 시뮬레이션을 시작할 수 있어요.`
- **UI 컴포넌트:**
  - 지역 카드 리스트 (가로 스크롤/선택 구조): `거제도` / `강화도` / `광양` (각 지역별 실사 배경 이미지 적용)
  - 하단 고정 버튼: `다음` (메인 테마인 그린 컬러 적용)

### 1.3. 3단계 온보딩 - [1] 기본 정보 입력
- **상단 인디케이터:** 3단계 스텝바 중 `1 기본 정보` 활성화, 뒤로가기 버튼(`<-`) 제공
- **화면 A (생년월일 입력):**
  - 타이틀: `태어난 날을 알려주세요.`
  - 서브: `만 14세 이상 사용자만 서비스 이용이 가능합니다.`
  - 입력 필드: `연도 / 월 / 일` 플레이스홀더 형태의 텍스트 필드
- **화면 B (성별 선택):**
  - 타이틀: `성별을 선택해주세요`
  - 컴포넌트: `남성` / `여성` 2분할 토글 버튼 (선택 시 연한 그린 배경 활성화)
- **화면 C (관심사 선택):**
  - 타이틀: `관심사를 알려주세요.`
  - 서브: `취미를 통해 적합한 지역을 찾아드려요.`
  - 칩(Chip) 컴포넌트 분류:
    - **라이프스타일 & 여가:** 영화, 연극/뮤지컬(선택), 독서, 전시, 음악, 여행, 맛집투어, 뜨개질, 캠핑, 반려동물, 공예
    - **스포츠 & 액티비티:** 러닝, 바이크, 야구경기관람, 축구, 테니스, 명상/요가
    - 기능 버튼: `+ 직접 추가하기`

### 1.4. 3단계 온보딩 - [2] 성향 및 가치관 설정
- **상단 인디케이터:** 3단계 스텝바 중 `2 성향 및 가치관` 활성화
- **화면 A (밸런스 게임형 성향 선택):**
  - 타이틀: `무엇이 더 좋나요?`
  - 서브: `성향에 맞는 레지던스, 귀촌 콘텐츠를 추천해드려요.`
  - 선택 카드쌍 (아이콘 포함 슬라이드/토글 구조):
    - `산` vs `바다`
    - `걷기` vs `차 타기`
    - `조용한 분위기` vs `활기찬 분위기` (각각 선택된 항목에 그린 컬러 하이라이트)
- **화면 B (인생 가치관 다중 선택):**
  - 타이틀: `00님께 중요한 삶의 가치는 무엇인가요? 마음에 닿는 걸 여러 개 골라주세요`
  - 서브: `5개까지 선택할 수 있어요.`
  - 칩(Chip) 컴포넌트 목록: 돈(선택), 건강, 시간, 자기계발, 사회적 인정(선택), 사랑, 취미생활, 자기관리, 봉사/나눔, 종교/신앙, 도전, 안정감, 자기 표현, 사회적 영향력, 가족, 권력(선택), 우정, 공동체
  - 기능 버튼: `+ 직접 추가하기`

### 1.5. 3단계 온보딩 - [3] 미래 그리기 (주관식 & 시나리오)
- **상단 인디케이터:** 3단계 스텝바 중 `3 미래 그리기` 활성화
- **화면 A (선호 상황 선택):**
  - 타이틀: `다음 중 00님이 선호하는 상황은 무엇에 가장 가까운가요?`
  - 객관식 라디오 버튼 리스트:
    - [V] `시장 아주머니들과 친해진다` (체크 및 그린 배경 활성화)
    - [ ] `한달살이를 시작한 이웃과 교류한다`
    - [ ] `아무도 없는 혼자만의 아지트를 찾아본다`
    - [ ] `또래를 찾아 취미를 공유한다`
- **화면 B (힐링 정의 선택):**
  - 타이틀: `마음 편한 '힐링'은 무엇에 가장 가까운가요?`
  - 객관식 라디오 버튼 리스트:
    - [V] `집에서 편히 쉬기` (체크 및 그린 배경 활성화)
    - [ ] `멋진 풍경을 감상하기`
    - [ ] `찐 로컬 맛집 탐방하기`
    - [ ] `새로운 사람들과 친해지기`
    - [ ] `최적의 공간에서 취미활동 하기`
- **화면 C (지역 특징 주관식 입력):**
  - 타이틀: `00님이 살고싶은 지역의 특징을 간단히 설명해주세요.`
  - 입력 필드: 텍스트 박스 (플레이스홀더: `예: 바다가 보이는 곳, 숲이 울창한 곳 ...`)

## 2. 메인 시뮬레이션 및 인터랙션 (Simulation Main)

### 2.1. 진입 연출 화면
- **카피라이팅:** - `반가워요, 00님`
  - `그럼, 꿈꾸던 지역에서 시작하는 삶을 살펴볼까요?`
- **UI 컴포넌트:** 사용자 캐릭터 아바타 에셋 중심 배치

### 2.2. 메인 지도 탭 (MAP Tab)
- **하단 내비게이션 바:** 5개 탭 구조 (`1.홈/지도(활성)`, `2.통계/차트`, `3.가계부/자산`, `4.로그/기록`, `5.프로필`)
- **상단 헤더:** `<- MAP`, 지역 태그(`강화도`), 진행 일차(`1일차`), 유저 프로필 아바타 및 이름(`김청풍 님`)
- **메인 영역 (달력 및 지도 인터랙션):**
  - **캘린더 드롭다운:** `2026.05 ∨` 클릭 시 주간 달력 뷰 노출 (일~토 요일 표시 및 현재 날짜 `19` 하이라이트)
  - **대한민국 전도 그래픽:** 시뮬레이션 진행 중인 행정구역이 컬러풀한 파스텔톤 레이어로 시각화됨. 강화도 인근에 펄스(Pulse) 형태의 로케이션 핀 표시.
  - **플로팅 포토 레이어:** 특정 일자나 이벤트 선택 시, 지도 위에 폴라로이드 사진 스타일의 귀촌 생활 아일랜드 컷(예: 농사 짓는 모습, 시골 풍경)이 겹쳐서 나타나는 인터랙션.
- **하단 카드 컴포넌트 (최근 레지던스 기록):**
  - 슬라이더 형태로 배치된 활동 카드.
  - 텍스트 구성: `최근 레지던스 기록`, 활동명(`고구마캐기`), 일자(`2026.05.02`), 장소(`강화`), 소요시간(`1h 13m`), 관련 수치(`11`, `37`)
  - 인터랙션: 카드 선택 시 주황색 테두리 하이라이트 및 체크 마크(`V`) 활성화 보완.

## 3. 다이어리 로그 및 퀘스트 (Log & Mission)

### 3.1. 레지던스 기록 관리 (4번째 탭)
- **상단 프로필:** 아바타 이미지, 현재 유저 등급 및 레벨 (`레지던스 Lv.1`)
- **중앙 세그먼트 탭:** 사용자가 경험한 지역 필터링 (`거제도`, `강화도(활성)`, `광양`)
- **카테고리 원형 아이콘 필터:** `생활비`, `의료`, `교통`, `인간관계`, `교육` 등 항목별 데이터 인덱싱
- **콘텐츠 리스트:** - `강화의 기록` 피드 카드 레이아웃
  - `프로젝트 일지` 카드 레이아웃
- **하단 플로팅 액션 버튼:** `오늘의 미션 수행하기` (그린 컬러 라운드 바 형태)

### 3.2. 지역 프로젝트 / 미션 수행 페이지
- **상단 비주얼 영역:** 그라데이션 배경과 3D 오브젝트 그래픽 적용.
  - 타이틀: `강화도편 잠시섬 프로젝트 참여`
  - 버튼: `참여 현황 보기`
- **퀵 가이드 그리드 메뉴:**
  - `길 찾아보기`, `기록하기`, `항목 점검`, `미션 기록` 4가지 기능의 버튼 배치
- **레지던시 추천 섹션:**
  - 사용자 매칭 점수 기반 리스트 뷰:
    - `아삭아삭순무민박` / `99점` / `▲ 31명`
    - `감자청년모임` / `30점` / `▲ 6명`
    - `일꾼민박` / `18분` / `▼ 7분`

## 4. 분석 대시보드 및 리포트 (Dashboard & Report)

### 4.1. 실시간 현황 대시보드
- **상단 배너:** `DashBoard`, 지역 및 일차 표시(`강화도 1일차`), 유저 아바타
  - 알림 문구: `김청풍님의 강화도 라이프를 분석했어요. [현황 보기]` 와 함께 캐릭터/우체통 3D 그래픽 배치.
- **현황 보고 섹션 (유동적 카드 발생):**
  - **이벤트 카드 1:** `따뜻한 친절을 받았어요! 이웃 주민 친절도 | 30점` (드롭다운 구조, 하단에 `확인`, `무시`, `더보기` 액션 텍스트 제공)
  - **이벤트 카드 2:** `맛집탐방러 김청풍님에게는 중앙시장의 접근성이 유익해요!`
- **주목할 항목 섹션:** 귀촌 생활 스탯 트래킹 (`이웃 주민 55점` 등 점수 및 게이지 바 시각화)

### 4.2. 최종 결과 팝업 (강화도 리포트)
- **UI 구조:** 모달/팝업 형태의 블러(Blur) 배경 처리, 우측 상단 닫기(`X`) 버튼
- **종합 요약 텍스트:** `강화도에서의 잠시섬 프로젝트는 따뜻한 힐링 감성과 함께 강화의 음식을 맛보고, 즐길 수 있었던 경험이었습니다. 친절한 주민들과 산책하기 좋은 코스가 인상깊습니다..`
- **인사이트 수치 명세:**
  - `맛집 점수`: `80점`
  - `생활 만족도`: `+ 70점`
  - `이웃주민`: `+ 88점`
  - **`Total`**: **`+ 258점`**
- **최종 액션 버튼 컴포넌트:**
  - 메인 액션: `다른 레지던시 둘러보기` (그린 라운드 버튼)
  - 서브 액션: `홈으로 돌아가기` (아웃라인 버튼)

## 5. 추천 미션 및 수행 시스템 (Mission & Verification System)

### 5.1. 미션 메인 대시보드
- **상단 헤더 정보:** 진행 단계 레이블(`강화도 레지던스 2일차`)
- **타이틀:** `오늘의 추천 미션`
- **서브 타이틀:** `직접 확인해보면 강화도와 한층 더 가까워질 거예요.`
- **UI 컴포넌트:**
  - **[추천] 메인 미션 카드:** 상단에 `추천` 뱃지가 부착된 강조형 카드. 
    - 텍스트: `전통 시장 방문하기` / `30분` / 난이도 `★★☆`
    - 디자인 적용: 메인 그린 테마 배경 및 화이트 텍스트 세팅
  - **카테고리 세그먼트 필터 (칩 버튼):** 사용자가 원하는 유형의 미션을 필터링 (`전체(활성)`, `생활체험`, `관계형성`, `현실체크`)
  - **미션 리스트 뷰:** 하단으로 나열되는 카드 스크롤 영역.
    - 서브 미션 항목 리스트: `주민과 대화 3분하기` (상세 스탯: `20분`, 난이도 `★★☆`, `87명 참여`)
  - **사용자 지표 및 히스토리 영역 (하단 고정):**
    - 스탯 카드: 트로피 아이콘과 함께 `강화도력 상위 23.34%` 표시 (`2026-05-10 측정 결과` 데이터 아카이빙)
    - 로그 박스: 이전 수행 내역을 확인할 수 있는 `미션 기록` 컨테이너 박스
