# 디자인 역할 가이드 — 청풍 프로젝트

## 핵심 역할
- 파스텔톤, 주황+초록 메인컬러 유지
- 모바일 화면 설계 (375px 기준)
- 온보딩부터 미션까지 톤앤매너 일관성 유지

## 컬러 시스템
| 용도 | 컬러 | 비고 |
|------|------|------|
| 메인 포인트 | 주황 (#FF7043 계열) | CTA, 강조 요소 |
| 서브 포인트 | 초록 (#66BB6A 계열) | 자연, 귀촌 느낌 |
| 배경 | 파스텔 크림/베이지 | 따뜻하고 부드러운 톤 |
| 텍스트 | 다크 브라운/차콜 | 가독성 확보 |

## 화면별 톤앤매너 기준
1. **온보딩** — 따뜻하고 설레는 느낌, 질문 카드 파스텔 배경
2. **홈(지도/피드)** — 밝고 자연스러운 톤, 지도에 초록 포인트
3. **레지던스 추천** — 카드형 UI, 사진 중심, 여백 충분히
4. **잠시섬 미션** — 경쾌하고 도전적인 느낌, 주황 포인트
5. **기록/아트맵** — 감성적, 갤러리 느낌, 여백과 텍스처 활용
