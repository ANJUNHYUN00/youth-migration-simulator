# 청풍 프로젝트

## 프로젝트 한 줄 요약
귀촌을 고민하는 사람들이 레지던스 프로그램을 체험하고 기록할 수 있는 모바일 웹 서비스

## 핵심 플로우
온보딩(취향 분석) → 홈(지도/피드) → 레지던스 추천 → 잠시섬 미션 수행 → 기록/아트맵

## 기술 스택
- 모바일 웹 (반응형, 375px 기준)
- 디자인 톤: 파스텔톤, 주황+초록 메인컬러

## 현재 진행 상황
- [x] 화면 기획
- [x] Claude Code 세팅
- [ ] 화면 데모 개발
- [ ] 잠시섬 미션 구성
- [ ] 아트맵 전시 시연

## 팀 역할 및 개발 순서
- 월: 준현 / 화: 기수 / 목: 나금 / 금: 현지
- 작업 시작 전 agents/ 폴더의 해당 역할 파일 읽기

## 개발 규칙
- 모바일 first
- 컴포넌트마다 한국어 주석
- 커밋 전 모바일 화면 확인
